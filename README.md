## Getting Started

This project require NodeJS

### Install

Clone this project

```bash
> git clone https://github.com/ArugaZ/whatsapp-bot.git
> cd whatsapp-bot
```

Install the dependencies:

```bash
> npm install
```

### Usage

Run the Whatsapp bot

```bash
> npm start
```
